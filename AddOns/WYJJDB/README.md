#忘忧景久素材库-WYJJDB

**WYJJDB** 是一个基于LibSharedMedia的魔兽世界共享媒体资源库插件。  
插件不包含任何玩法或逻辑功能，仅用于向游戏中注册可供 UI 插件使用的媒体素材。

### 包含内容
- 忘忧景久录制的简短语音提示  
  - 语音内容与灵感致谢：Causese
- Ethric Arrow Plater目标指示箭头  
  - 来源：https://wago.io/okQgj8zoy
- 原生风格进度条材质  
  - 来源：https://wago.io/j0VJqpjmc

安装后，所有素材将出现在支持LibSharedMedia的插件下拉菜单中  
（如 WeakAuras、Plater、ElvUI 等），可直接试听和选择使用。

---

**WYJJDB** is a World of Warcraft addon that extendsLibSharedMedia by registering additional shared media assets.  
It contains no gameplay or UI logic and functions purely as a media library.

### Included Media
- Short custom voice clips recorded by Wangyou Jingjiu  
  - Voice content and inspiration with special thanks to Causese
- Ethric Arrow target indicator for Plater  
  - Source: https://wago.io/okQgj8zoy
- Blizzard-style status bar textures  
  - Source: https://wago.io/j0VJqpjmc

All assets will be available in the dropdown menus of addons that support LibSharedMedia,  
such as WeakAuras, Plater, and ElvUI.
